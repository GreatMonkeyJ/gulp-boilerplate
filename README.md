## Build Setup

``` cli
# install dependencies
npm i

# serve with hot reload at localhost:8080
npm run serve

# build for production
npm run build

# deploay for production images
npm run deploy

# clean working directory
npm run clean:serve

# clean build directory
npm run clean:build

```
